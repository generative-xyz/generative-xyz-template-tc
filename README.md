## How to use

# Install

```git clone https://github.com/fxhash/fxhash-webpack-boilerplate.git```

# Start local

```npm start```

# Build

```npm build```

Will build your code into ```/dist``` with ```index.html, index.js, styles.css``` and zip all to ```project.zip```
